*Mälardalen University · Oct 2025 – Jan 2026*

Redesigned the user interface of the INSIGHTS personal development micro-learning app (iOS) to improve usability, engagement, and learning retention — moving users from passive reading to active reflection. Led the project end-to-end as a solo designer using a user-centred design approach: research, workshops, and iterative prototyping.

**Outcomes**

- Simplified navigation and interactions to reduce cognitive load
- Card-based swipe interaction for faster, more intuitive content browsing
- Reflection diary feature to help users actively apply what they learn
- Daily streak system to encourage consistent usage and habit building
- Validated through user testing; delivered a high-fidelity Figma prototype
