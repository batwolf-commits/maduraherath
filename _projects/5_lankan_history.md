*SLIIT · Jan 2022 – Apr 2022*

Developed "Lankan History," a location-based Android app to help users discover nearby historic sites, hotels, and vacation rentals across Sri Lanka, with an intuitive filtered search, navigation, and real-time transit info to enhance local and international tourism experiences.

**Key features**

- Integrated Google Places API and Google Directions API for directions and distance mapping
- Filtered search for historic places and nearby accommodations using A-GPS location
- Secure MySQL cloud database for user preferences and login credentials
- Six primary app screens including landing, sign-in/sign-up, nearby search, and account settings
